Artículo 1 — Diseño Web Responsivo

El diseño web responsivo permite que un sitio se adapte a diferentes tamaños de pantalla y densidades de píxeles, garantizando una experiencia consistente y usable. En la actualidad, los usuarios acceden desde teléfonos, tablets, laptops y pantallas 4K; un diseño adaptable mejora la legibilidad, la navegación y el rendimiento percibido.

  • Enfoque mobile-first y contenido flexible.
  • Imágenes y tipografías escalables.
  • Breakpoints con media queries para ajustar el layout.

Artículo 2 — Viewport

El viewport define el área visible de una página web en el dispositivo. En móviles, es esencial configurar la metaetiqueta para que el ancho lógico coincida con el ancho del dispositivo y el nivel de zoom inicial sea adecuado. 

<meta name="viewport" content="width=device-width, initial-scale=1">

Con este ajuste, las unidades relativas (%, vw, rem) y los breakpoints se comportan como se espera en distintos dispositivos.

CSS3 Media Queries

Las media queries permiten cambiar estilos según el ancho de la ventana, la orientación o la densidad de píxeles. Ejemplo práctico (mobile-first): 

/* Base (móvil) */
.card-demo { font-size: 1rem; }

/* ≥ 576px */
@media (min-width: 576px) {
  .card-demo { font-size: 1.05rem; }
}

/* ≥ 768px */
@media (min-width: 768px) {
  .card-demo { font-size: 1.1rem; }
}

/* ≥ 992px */
@media (min-width: 992px) {
  .card-demo { font-size: 1.15rem; }
}
Este texto aumenta levemente de tamaño a medida que la pantalla es más ancha.

.grid-sample {
  display: grid;
  grid-template-columns: repeat(2, 1fr);
  gap: .75rem;
}
@media (min-width: 768px) {
  .grid-sample { grid-template-columns: repeat(3, 1fr); }
}
@media (min-width: 992px) {
  .grid-sample { grid-template-columns: repeat(6, 1fr); }
}

Bootstrap 5: Componentes Clave

Containers

Los contenedores .container y .container-fluid establecen anchos máximos y paddings horizontales. Este sitio usa .container-lg para el contenido. 

<div class="container">...</div>
<div class="container-fluid">...</div>

Grid System

El sistema de cuadrícula de Bootstrap divide la fila en 12 columnas. Ejemplo: 

<div class="row">
  <div class="col-12 col-md-6">Col A</div>
  <div class="col-12 col-md-6">Col B</div>
</div>

Flex

Las utilidades Flex de Bootstrap agilizan alineaciones y distribuciones.

.d-flex .justify-content-between .align-items-center .flex-wrap 
<div class="d-flex justify-content-between align-items-center">...</div>

Carousel

<div id="demoCarousel" class="carousel slide" data-bs-ride="carousel">...</div>

Navbar

La barra de navegación superior usa .navbar, colapsa en móvil y expande en escritorio.
